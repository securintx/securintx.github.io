## Devlopment & Content Writing

Do not use `main` branch use `dev` branch and create **pull-request** for update

```bash
# copy .env.example to .env
$ cp .env.example .env

# modify .env
$ nano .env

# run hugo site
$ docker-compose up -d serve
```
